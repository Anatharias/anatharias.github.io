i Que contient le fichier Profil JSON ?
⚠️ Données directement nominatives présentes
Ces informations permettent d'identifier l'utilisateur de façon explicite :
  • Nom d'utilisateur Plex — champ "user" (ex. "user": "Anatharias")
  • Nom d'utilisateur Discord — champ "discord_name" (ex. "discord_name": "Anatharias")
  • Nom du fichier — contient le pseudo et la date de génération (ex. anatharias_20260325.json)
✅ Ce qui n'est PAS présent dans le fichier
  • Aucune adresse IP
  • Aucun email
  • Aucun identifiant technique Plex (UUID de compte, token, etc.)
  • Aucune donnée de localisation
💬 Pourquoi le pseudo Discord est indispensable
Le nom d'utilisateur Discord doit obligatoirement être renseigné dans votre profil pour que la communauté puisse vous identifier lors des échanges.

Sans lui, il est impossible de savoir à qui adresser une demande de partage réciproque — c'est l'unique lien entre votre profil Plex partagé et votre identité sur notre serveur Discord.

👉 Renseignez votre pseudo Discord avant d'exporter votre profil JSON pour le partager avec la communauté.
ℹ️ Guide des boutons du tableau de bord
← Exporteur
Retourner à l'exporteur
Revient à l'interface principale de l'exporteur Plex, là où vous pouvez connecter votre serveur et lancer un export CSV.
🗑 Clear
Effacer toutes les données
Supprime toutes les données chargées en mémoire (films, séries, musique, docs). Le dashboard revient à son état vide. Attention : irréversible sans re-importation.
📂 Import
Importer un fichier CSV ou JSON
Ouvre le sélecteur de fichier pour charger vos données dans le dashboard. Accepte deux formats :
• CSV — export complet généré par l'exporteur Plex (un nom Discord vous sera demandé)
• JSON Profil — fichier de profil exporté par ce dashboard
💾 Export CSV
Télécharger les données au format CSV
Re-télécharge les données actuellement chargées sous forme de fichier CSV. Utile pour sauvegarder l'état courant ou l'ouvrir dans un tableur (Excel, Google Sheets, etc.).
🔗 Export Profil JSON
Générer et partager votre profil enrichi
Crée un fichier JSON compact contenant votre bibliothèque complète + des statistiques détaillées (résolutions, codecs, notes, durées, stockage, genres dominants…). Ce fichier est automatiquement uploadé sur Firebase pour être partagé avec la communauté. Cliquez sur le bouton i à côté pour en savoir plus sur les données incluses.
📸 Capturer PNG
Sauvegarder le dashboard en image
Génère une capture d'écran PNG des tuiles et graphiques principaux du dashboard. Pratique pour partager un aperçu visuel de votre bibliothèque sur Discord ou les réseaux sociaux.
